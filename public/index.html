Инструкция по подключению:

  • 1. Перейдите в телеграм бота https://t.me/nishinskogo38svet_bot и нажмите Start.
  • 2. Напишите /status чтобы узнать свой chatid.
  • 3. Подключитесь к вашему OpenWRT роутеру и по SSH.
  • 4. Откройте файл:
  • vi /usr/bin/ping.lua
  • Вставьте следующий код:
  • while true do os.execute("wget -O- 'https://wrtping.glitch.me/ping?chat_id=ваш_chatid'") os.execute("sleep 60") end
  • Чтобы скрипт запускался автоматически при старте роутера
  • vi /etc/rc.local
  • Добавьте строку для запуска вашего скрипта перед строкой exit 0.
  • Это должно выглядеть так:
  • #!/bin/sh # This script is executed at the end of the boot process. # You can add your own commands below. lua /usr/bin/ping.lua & exit 0
  • Сохраните изменения и выйдите из редактора. Нажмите Esc затем написать :wq
